1 · THE QUESTION SPLITS

The engine reads the headlines and the base rates, then splits into eight independent runs. No sharing, no groupthink: each run sees the future for itself.

2 · WORLDS BRANCH

Each run imagines the event playing out again and again. Hot blue worlds resolve YES, cool red worlds NO, like real stars. The spread between them is the engine's doubt.

3 · REALITY GRADES IT

Every pick is logged and scored against real closing prices. The rules for "actually good" were locked before any results existed.
